Herança entre classes
-É um conceito da programação orientada a objetos que permite que uma classe reutilize os membros (como métodos e propriedades) de outra classe.{A classe que herda é chamada de classe derivada, e a que é herdada é a classe base.}

Modificador de acesso 
	[Protected]
	Define que um membro da classe só pode ser acessado dentro da própria classe e por classes que a herdam. É mais restrito que public, mas mais aberto que private.
	[Abstract]
	Indica que uma classe ou método é incompleto e precisa ser implementado por uma classe derivada. Classes abstratas não podem ser instanciadas diretamente.
	[Virtual]
	Permite que um método seja sobrescrito em uma classe derivada. É usado para definir um comportamento padrão que pode ser modificado.
	[Sealed]
	Impede que uma classe seja herdada ou que um método sobrescrito seja novamente sobrescrito. Serve para limitar a extensão de herança ou modificação.
